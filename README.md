Класс Graphic визуализирует построение кратчайшего пути между двумя точками на поле с препятствиями. Препятствия представляют собой непересекающиеся полигоны без самопересечений.
Нажатиями мышки на экран строятся полигоны. Последний отрезок полигона, соединяющий последнюю точку с первой, строится нажатием Space, после чего можно снова начать построение нового полигона нажатиями мышки. 
Когда все препятствия расставлены, следует нажать Enter и двумя щелчками мыши расставить точки, путь между которыми требуется найти.
